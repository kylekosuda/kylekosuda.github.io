Theme Park Safety Control Prototype

This project is a real-time safety control prototype for a theme park ride, developed in Wokwi using an ESP32 microcontroller and FreeRTOS. The system monitors simulated ride restraint pressure, provides a manual emergency stop button, and activates visual alerts when unsafe conditions are detected. A green status LED indicates normal operation, while a red LED signals an active emergency stop. The design uses tasks, interrupts, semaphores, and a mutex to demonstrate predictable timing and reliable response for safety-critical events.
